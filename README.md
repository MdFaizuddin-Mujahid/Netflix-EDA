Perform Exploratory Data Analysis (EDA) on a movie and TV show dataset to discover trends and insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
mymoviedb.csv

## Analysis Performed
- Data Cleaning
- Missing Value Analysis
- Descriptive Statistics
- Genre Analysis
- Rating Analysis
- Visualizations
